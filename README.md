# The-Path-of-Truth

# Dhammapada 2600: The 26 Essentials ☸️

โครงการรวบรวม "หัวใจ" หรือแก่นธรรมจากพระธรรมบท (Dhammapada) ทั้ง 26 วรรค เพื่อฉลองวาระพุทธชยันตี 2,600 ปีแห่งการตรัสรู้ โดยเน้นการสรุปเนื้อหาให้กระชับ เข้าใจง่าย และนำไปประยุกต์ใช้ได้จริงในชีวิตประจำวัน

## 🌟 จุดเด่นของโปรเจกต์
- **The Gist:** สรุปใจความสำคัญของแต่ละวรรค (Vagga) ไม่ต้องตีความซับซ้อน
- **Minimalist Design:** หน้าเว็บอ่านง่าย สบายตา สไตล์สะอาดสะอ้าน
- **Responsive:** รองรับการอ่านทั้งบนคอมพิวเตอร์ แท็บเล็ต และสมาร์ทโฟน
- **Single File:** โครงสร้าง HTML/CSS จบในไฟล์เดียว สะดวกต่อการนำไปใช้งานหรือส่งต่อ

## 📖 สารบัญเนื้อหา (The 26 Vaggas)
1. **ยมกวรรค** - หมวดคู่ (ใจเป็นประธาน)
2. **อัปปมาทวรรค** - หมวดความไม่ประมาท
3. **จิตตวรรค** - หมวดจิต
... (และหมวดอื่นๆ จนครบ 26 วรรค)

## 🚀 วิธีการใช้งาน
1. Download หรือ Copy ไฟล์ `index.html`
2. เปิดไฟล์ด้วย Web Browser (Chrome, Firefox, Safari ฯลฯ)
3. เริ่มต้นการเดินทางแห่งปัญญาด้วยการอ่านวันละหนึ่งวรรค

## 🕊️ วัตถุประสงค์
เพื่อเผยแผ่พระธรรมคำสอนในรูปแบบที่เข้ากับยุคสมัยดิจิทัล และเป็นธรรมทานแก่ผู้ที่สนใจศึกษาแก่นแท้ของพุทธศาสนสุภาษิต
